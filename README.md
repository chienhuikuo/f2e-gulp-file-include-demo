# This is Gulp package gulp-file-include demo

## requirements

1. nodejs >= 12.16.0
2. yarn >= 1.22.0
3. gulp >= 4

## init

```bash
yarn global add gulp-cli
yarn install
```

## tasks

### clean dist files

```bash
gulp clean-dist
```

### do js lint

```bash
gulp js-lint
```

### build files

```bash
gulp build
```

### run live server and reloading

```bash
gulp live
```
